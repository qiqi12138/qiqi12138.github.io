Node.js - 使用 Node 搭建一个服务器

October 3, 2018

使用 Node 搭建一个服务器

  • 在 Node 中专门提供了一个核心模块http

  • http这个模块的职责就是帮你创建编写服务器的

第一步 加载 http 模块

var http = require('http');

第二步 创建 Server

var server = http.createServer();

  • 使用 http.createServer() 方法创建一个 web 服务器

  • 该方法返回一个 Server 实例

第三步 监听 request 响应事件,设置请求处理函数

server.on('request', function (request, response) {  
    console.log('收到客户端的请求了, 请求路径是:' + request.url)
    // 发送请求头 用来解决中文乱码
    response.writeHead(200, { 'Content-Type': 'text/plain;charset=utf-8' })
    // 给客户端发送一条响应数据
    response.write('hello NodeJS')
    // 告诉客户端 我的话说完了可以呈现给客户端了
    response.end()
})
  • request 请求事件处理函数,需要接收两个参数
    • Request 请求对象
    • 请求对象可以用来获取客户端的一些请求信息. 例如请求路径 request.url
    • Response 响应对象
    • 响应对象可以用来给客户端发送响应消息. 例如发送请求头,用来解决中文乱码 response.writeHead(200, { 'Content-Type': 'text/plain;charset=utf-8' })
  • response 对象有一个方法: response.write() 可以用来给客户端发送响应数据
    • response.write() 可以使用多次, 但是最后一定要使用 response.end() 来结束响应, 否则客户端会一直等待

第四步 绑定端口号，启动服务器

server.listen(3000, function () {  
    console.log('服务器启动成功了,可以通过 http://127.0.0.1:3000/ 来访问')
})
